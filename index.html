    सिया पुस्तकालय सफ्टवेर

    [Currently Being Hosted At Nepal Japan Children Library, Lainchaur, Kathmandu]
    Siya is a complete library management system which provides the following features
    • Book Cataloguing System
    • Easy Book Circulation
    • Multi User Environment
    • Create Permissions for user and groups
    • Dynamic Fields for books
    • Ajax based Data Entry System
    • Fuzzy Search Implementation to find books
    • Easily add users, groups and create permissions

    What Can Siya Do?

    • Catalog Books
    • Manage Members, volunteers and librarians
    • Lend books, return books
    • Take care of your accounting journal, ledger and trial balances
    • Search For Books based on all their properties

    Objectives

    Siya started as a few scripts I wrote to automate a few tasks at a library I was volunteering at. Over the span of a few months, Sia grew to become a fully functioning library management system. While, developing Sia, I had the following guidelines in mind (Some of them are rip-offs from the zen of python)

    • Easy to use by volunteers with minimal training
    • Should support Nepali as it's default language
    • Should be simplistic in design
    • There should only be one obvious way to do something
    • If the implementation is hard to explain, it's a bad idea.
    • Explicit is better than implicit.

    Sia Library Software is built using the following projects

    • Django Web Framework
    • JQuery
    • pyBSDate

    How To Install Siya

    To install on a debian system, run this : 

    wget -qO- https://raw.githubusercontent.com/ayys/alms/master/install.sh | sh

    The above code will start the server and run it in the default browser.

    To start the server again, simply go to the alms directory and run start.sh
